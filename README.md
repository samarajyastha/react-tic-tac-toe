# Tic Tac Toe

Play the game [Tic Tac Toe](https://react-tic-tac-toe-x.herokuapp.com/)

## Rules

- Both players has uses 'X' to play.
- Whoever makes 3 in a row loses the game.
- The key is to avoid making a row.
